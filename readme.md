## Problem
I have a blog with 5000 articles. I need to implement search features that will search by article title. How can I implement this in real time without any delays.

## About this project
I tried to solved the problem above. To improve the search speed, I applied Cache of Laravel. Also by tracking console.time(), I applied some change in the client side to improve the speed of showing result. The change helped search feature become nearly real time.
